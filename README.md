# Project Title
This project is about determining if a random number is positive, negative, or zero.

## Requirements
- gcc
- Make

## How to run
1. Clone the repository
2. Compile the program: `gcc -Wall -pedantic -Werror -Wextra -std=gnu89 0-positive_or_negative.c -o 0-positive_or_negative`
3. Run the program: `./0-positive_or_negative`
