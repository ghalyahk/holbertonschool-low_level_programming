# Holberton School Low-Level Programming

## Description
This repository contains my solutions for the **Doubly Linked Lists** project as part of the Holberton School Low-Level Programming curriculum.  
It includes C programs that implement and manipulate doubly linked lists.

## Files
- `lists.h` - Header file containing the doubly linked list structure and function prototypes.
- `0-print_dlistint.c` - Prints all elements of a doubly linked list.
- `3-add_dnodeint_end.c` - Adds a new node at the end of a doubly linked list.
- `4-free_dlistint.c` - Frees a doubly linked list.
- `5-get_dnodeint.c` - Returns the nth node of a doubly linked list.
- `5-main.c` - Main file to test the functions.

## Compilation
Compile the files using:

```bash
