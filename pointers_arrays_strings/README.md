# Pointers, Arrays and Strings

This directory contains C programs focused on:

- Understanding pointers and how they work.
- Manipulating arrays using pointers.
- Working with strings in C.
- Practicing function definitions and prototypes.

## Files

| File | Description |
|------|-------------|
| `6-puts2.c` | Prints every other character of a string, starting with the first character. |
| `6-main.c` | Main function to test `puts2` function. |
| `_putchar.c` | Custom implementation of `_putchar` function. |
| `main.h` | Header file with function prototypes. |

## Compilation

All programs are compiled with:

```bash
gcc -Wall -pedantic -Werror -Wextra -std=gnu89 <file>.c -o <output>
