# Variadic Functions in C

This project demonstrates the use of **variadic functions** in C.

## File Descriptions

- `0-sum_them_all.c`: Contains a function that returns the sum of all its parameters.
- `variadic_functions.h`: Header file containing the function prototype.
- `0-main.c`: Test file to check the functionality (provided).
